HTML Section

  • Hypertext Markup Language (HTML) is one of the three cornerstone languages used in web development. HTML is used to create the structure of a webpage.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • Using HTML is like building a sturdy "frame" for your website.
  • The head element contains information about a webpage.
  • The body element represents the visible content shown to the user.

CSS Section

  • Cascading Style Sheets (CSS) is another cornerstone language used in web development to describe the presentation of a website thats created with HTML and how each element on a webpage is rendered.
  • CSS is used for the styling and layout of a webpage. This includes, altering the font, the color, the size, and the spacing throughout your content.
  • There are three different ways to style a webpage using CSS:
    • Inline CSS: used when applying a specific style to a single HTML.
    • Internal CSS: used to define a style for a single HTML.
    • External CSS: is a seperate file and is used to style the HTML.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git Section

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creates a new branch and switches to it.

Javascript Section

  • Javascript is the last of the three cornerstone languages used in web development.
  • Javascript is a lightweight programming language that is used to provide logic and interactivity for websites.
